About

The engineer behind the code

I'm a third-year B.Tech Computer Science student at Graphic Era Hill University, Dehradun. I build full-stack products and AI systems — currently interning as an AI-Assisted Full Stack Developer at TBI-GEU and an AI Intern at Mirai School of Technology, alongside AWS certifications in Cloud and AI.

Outside of internships, I ship my own projects end to end — design, backend, deployment — because I'd rather learn from a live app with real users than a tutorial.

"I like building things that admit what they don't know. SheSays exists because a comment box shouldn't feel like a courtroom. VibeOps exists because I got tired of manually relaying context between AI tools — so I built the relay instead. ATSlay exists because a candidate deserves to see exactly which keywords are missing, not a mysterious black-box score. Every project I ship documents its own limits — pretending v1 is perfect is the fastest way to stop improving it."
